1. Welcome

This Setup Wizard will assist you with setting up your RavenDB server.
First, select whether to set up a new cluster from scratch or continue the setup using an existing Setup Package.

New Cluster

When to use:
  • Setting up a new single-node cluster
  • Setting up a new Cluster:
    • This Setup Wizard will configure the cluster and set up the first node.
    • After that, run the Setup Wizard again per node and select option: 'Use Setup Package'.
  • Optionally, create a Setup Package for an external environment
    i.e. cloud instance, containers, or similar.

Use Setup Package

When to use:
  • Only after you have obtained a Setup Package
  • Setting up a new cluster
  • Setting up another node in an existing cluster
To disable this wizard and setup the server manually:
  • Open the settings.json file located in your RavenDB installation directory
  • Change the setup mode to None, e.g. "Setup.Mode: "None"
